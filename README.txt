OneDriveSyncHelper
=================

A simple tool to force OneDrive synchronization.

Building on Windows:
--------------------

1. Install Qt 5.15.2 or newer from https://www.qt.io/download-open-source
2. Install CMake from https://cmake.org/download/
3. Make sure both Qt and CMake are in your PATH
4. Run build_windows.bat and follow the instructions

Using the application:
---------------------

1. Run OneDriveSyncHelper.exe
2. Right-click on the system tray icon to access options
3. Set your OneDrive path in the settings
4. Use "Sync OneDrive" to trigger synchronization
